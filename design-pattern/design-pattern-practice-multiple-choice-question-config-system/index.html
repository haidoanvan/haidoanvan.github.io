Luyện tập Design Pattern: Hệ Thống Cấu Hình Câu Hỏi Trắc Nghiệm

March 23, 2023

Bạn cần thiết kế một hệ thống quản lý câu hỏi trắc nghiệm cho một nền tảng học trực tuyến. Hệ thống cần hỗ trợ nhiều loại câu hỏi như:

  • Câu hỏi một lựa chọn.
  • Câu hỏi nhiều lựa chọn.
  • Câu hỏi điền vào chỗ trống.

Yêu cầu:

  • Dễ dàng mở rộng để thêm các loại câu hỏi mới.
  • Hỗ trợ hiển thị câu hỏi trên nhiều giao diện khác nhau.
  • Hỗ trợ kiểm tra đáp án và tính điểm tự động.
